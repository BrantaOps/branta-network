# Branta Network

Once your platform is integrated with Branta, open a PR to be added to https://www.branta.pro/network.

### Wallets

Format: 
```
    {
      "name": "My Wallet Name",
      "logo": "https://example.com/logo.svg",
      "website_url": "https://example.com"
    }
```
